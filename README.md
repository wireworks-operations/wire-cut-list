# EECOL Wire Cut List 🗂️

![EECOL Wire Cut List Icon](icon.png)

A professional management tool for tracking and organizing wire cut orders, specifically designed for EECOL Electric.

## Features

- **Fuzzy Search**: Quickly find orders by number, customer, or wire type using intelligent fuzzy matching.
- **Advanced Filtering**: Filter items by Status (Active, Completed, Archived) and Entry Type (IBT, Customer Order, etc.).
- **Urgency Visualization**: Items are visually distinguished by urgency (Urgent, High, Normal) with color-coded indicators.
- **Bulk Actions**: Select multiple items to perform batch operations like marking as complete or archiving.
- **Dynamic Sorting**: Sort the list by Order Number, Customer, Urgency, or Date Created.
- **Drag-and-Drop**: Manually reorder items in the list for custom prioritization.
- **Real-time Updates**: Real-time feedback and toast notifications for all actions.
- **Persistence**: All data is saved locally in your browser.

## Getting Started

1. Add items using the "+ Add Item" button.
2. Use the search bar for fuzzy matching across multiple fields.
3. Click on items to select them for bulk actions.
4. Right-click on an item for a context menu with more options (Edit, Change Color, Delete).
5. Use the filter and sort dropdowns to organize your view.

- **React 19**
- **Vite**
